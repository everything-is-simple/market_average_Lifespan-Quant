# 价格行为四书-分工与阅读顺序

日期：2026-03-28

## 结论先说

你把这三本放在一起：

- `Lance Beggs / YTC`
- `Bob Volman《外汇超短线交易》`
- `许佳冲《裸K线交易法》`

这件事我觉得是对的。

但如果再加上 `Al Brooks`，就不能四本平放。

更准确的关系应该是：

- `Lance + Bob + 许佳冲` 组成你的**执行层价格行为三角**
- `Al Brooks` 作为它们背后的**原典母本 / 语法总源**

一句话说：

**前三本适合拿来做“会做单”的训练，Al Brooks 更适合拿来做“理解价格行为语言”的底层参考。**

## 为什么你会反复读前三本

因为这三本都更接近“能落地”。

### 1. Lance Beggs

Lance 最强的是：

- 市场分析流程
- 架构区
- 预期
- 入场
- 止损
- 目标
- 交易管理

他把价格行为拆成了流程，所以最适合拿来做：

`执行框架`

### 2. Bob Volman

Bob 最强的是：

- 结构
- 突破
- 二次突破
- 箱体突破
- 区间突破
- 临界点技术
- 不利市场条件

他把超短交易做成了非常清晰的“局面学”。

所以他最适合拿来做：

`结构语法 + 节奏语法`

### 3. 许佳冲《裸K线交易法》

这本最强的是：

- 教学改写能力强
- 关键位讲得直
- Pinbar、信号、入场止损都压得很实用
- 复盘和心理补得很顺手

它像一本：

`中文执行教材 / 上手书`

所以你会觉得它很顺手，这是正常的。

## 为什么 Al Brooks 十年了都难懂

这也正常，而且不是你一个人的问题。

Al Brooks 难，不只是因为厚。

更根本的原因有四个：

### 1. 他不是在讲“形态”，而是在讲“语言”

很多书在教：

- 这个形态是什么
- 这个信号怎么做

Al Brooks 其实在做的，是把每一根 K 线、每一次测试、每一段趋势、每一次失败都看成语言单元。

所以读他时，会有一种感觉：

- 明明每句话都认识
- 但整段意思像在绕

因为他不是在给你口诀，而是在训练你“读价格句子”。

### 2. 他默认读者已经有很强的盘面经验

Al Brooks 很少像教学书那样一步步照顾初学者。

他经常默认你已经能看懂：

- 趋势棒
- 测试
- 失败突破
- 二次入场
- 交易区间
- 趋势与区间的相互转化

如果这些经验没在脑子里长稳，他就会显得特别重。

### 3. 他的信息密度极高，而且重复中带微调

Al Brooks 的特点不是“说一次说明白”，而是：

- 从不同角度反复说
- 每次只微调一点点

这对建立深层语感很有效，但对普通阅读很不友好。

所以会出现典型体验：

- 觉得在重复
- 又觉得每次不完全一样

这就是他难的原因之一。

### 4. 他更像原典，不像教程

Lance、Bob、许佳冲都更像“教你怎么做”。

Al Brooks 更像：

`价格行为原典 / 语法母本`

原典的特点就是：

- 概念源头很强
- 但不一定适合先拿来入门

## 四本最合理的分工

### 第一层：上手与训练

放：

- `许佳冲`

作用：

- 快速建立中文语境
- 把 `关键位 + 信号 + 入场止损 + 复盘` 先串起来

### 第二层：结构与节奏

放：

- `Bob Volman`

作用：

- 训练你对 `突破 / 区间 / 二次突破 / 临界点` 的感知
- 建立盘中结构的节奏感

### 第三层：执行框架

放：

- `Lance`

作用：

- 把前两层变成完整交易流程
- 让你知道什么时候分析、什么时候等、什么时候进、什么时候管

### 第四层：原典和母语

放：

- `Al Brooks`

作用：

- 负责给前三层提供更深的价格行为语法
- 用来解释那些你明明会做、但说不清为什么的细节

## 所以这四本不该怎么摆

不建议这样摆：

- 四本都算“价格行为书”
- 所以并列等权反复读

这样容易出问题：

- Lance 的流程感被削弱
- Bob 的超短结构和 Al 的原典语言混在一起
- 许佳冲的教学简化和 Al 的原典密度互相打架

## 更推荐的摆法

我建议你把它们摆成：

`Al Brooks`
`  -> 母本 / 词典 / 语法源`

`Lance`
`  -> 执行主框架`

`Bob`
`  -> 结构与突破节奏专精`

`许佳冲`
`  -> 中文教学改写与复盘入口`

也可以换一种更适合你系统的说法：

- `Lance = 主手册`
- `Bob = 超短结构专精册`
- `许佳冲 = 中文训练册`
- `Al Brooks = 原典参考书`

## 最推荐的阅读顺序

如果你已经读了前三本很多遍，而 Al Brooks 仍然难，我建议现在不要硬啃 Al 的全套顺读。

更有效的顺序是：

1. `许佳冲`
   先把中文关键位、信号、入场止损、复盘继续压实。
2. `Bob Volman`
   再把突破、区间、临界点、节奏强化。
3. `Lance 卷2/卷3/卷4`
   把一切收束到流程、架构区、执行和管理。
4. `Al Brooks`
   只做“定点查词典式阅读”。

也就是说，不要问：

- `今天把 Al Brooks 从头读到尾行不行`

而要问：

- `今天我在 Lance/Bob/许佳冲里卡住的那个点，Al Brooks 对应怎么说`

这样他会突然变得有用得多。

## Al Brooks 最适合怎么用

我建议你把 Al Brooks 当成：

`母本词典`

而不是：

`第一本入门书`

正确使用姿势：

1. 先在 Lance/Bob/许佳冲里形成一个你能操作的概念
2. 再去 Al 里找它更细、更原始的语言
3. 只定点攻克，不整本硬啃

比如你可以只定点看：

- 趋势棒与十字星
- 突破、测试、反转
- 二次入场
- 交易区间
- 趋势线与通道

这比从第一页硬顶到最后一页有效得多。

## 最后裁决

你真正该放在一起的，不是“价格行为四本等权书”，而是：

`三本执行书 + 一本原典母本`

其中：

- `Lance` 是主轴
- `Bob` 是结构强化
- `许佳冲` 是中文落地
- `Al Brooks` 是母本校准

一句话收口：

**前三本帮你会做，Al Brooks 帮你最终看懂自己为什么会做。**
