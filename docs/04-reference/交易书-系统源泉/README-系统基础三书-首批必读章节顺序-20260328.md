# 系统基础三书-首批必读章节顺序

日期：2026-03-28

## 一句话先讲

如果你真把这三本当系统基础，不要三本一起散读。

最有效的顺序是：

1. `《交易圣经》` 先立系统工程骨架
2. `《专业投机原理》` 再压实趋势、风险、心理与总框架
3. `Lance / YTC` 最后进入价格行为执行细部

## 第一部：《交易圣经》先读顺序

### 首批必读

1. `第1章 交易真相`
   先把“为什么十有九亏”钉死，把常见错误、资金管理、心理、方法的基本误区先拆掉。
2. `第2章 交易过程`
   这是整本书的流程总图，先知道系统交易到底是哪些环节拼起来的。
3. `第7章 原则五：三大支柱`
   这是全书总轴，直接把 `方法 + 资金管理 + 心理因素` 立为系统三柱。
4. `第8章 资金管理`
   这是系统层最硬的一章，不先吃下它，后面的方法章会失真。
5. `第9章 方法`
   重点看“趋势交易应当简单化”“支撑阻力就足够”“等待回调模式、进入信号、止损、管理、兑现利润”。
6. `第10章 心理因素`
   把系统工程里的心理层补齐，不然你会只剩结构没有执行力。
7. `第11章 原则六：交易`
   这是从方案落到指令与执行的桥梁章。

### 第二批再读

1. `第3章 准备`
2. `第4章 启蒙`
3. `第5章 交易风格`
4. `第6章 市场`

这几章更适合在你已经立住系统骨架后回头精读。

### 可后读

1. `第12章 大师卓见`
2. `第13章 结束语`

这些有价值，但不是你当前搭系统主线的第一优先。

## 第二部：《专业投机原理》先读顺序

### 首批必读

1. `第3章 一致性成功的事业经营哲学`
   这章最适合先抓，因为它直接把投机当成一门经营活动，而不是点子游戏。
2. `第4章 在混沌的市场中寻找秩序：道氏理论`
   这是总框架基础。
3. `第5章 趋势的定义`
   趋势到底是什么，这章必须先吃透。
4. `第7章 创造财富的契机：辨识趋势的改变`
   重点就是 `趋势线、1-2-3、2B、试探`，这对你当前系统非常关键。
5. `第11章 衡量风险以管理资金`
   风险和资本配置是这本书真正的硬骨头之一。
6. `第12章 让你发生亏损的方法至少有50种`
   这章很适合拿来当“失败学”补丁。
7. `第13章 史波克症候群：理性与情绪之间的战争`
   把情绪纪律正式压进系统。

### 第二批再读

1. `第6章 技术分析的优点与缺点`
2. `第8章 失败可能来自于分析报告未提供的资料`
3. `第26章 风险-回报分析的技术性基础`
4. `第27章 市场分析的技术性原则`
5. `第28章 技术分析原则的运用`

这几章很重要，但最好在前面的主骨架建立后再回来读。

### 可后读

1. `第1章 从赌徒到市场宗师`
2. `第9章、第10章、第18章以后的大量经济学与宏观部分`
3. `期权与当日冲销章节`

这些不是没价值，而是当前不该抢主线注意力。

## 第三部：Lance / YTC 先读顺序

### 首批必读

1. `卷2：市场和市场分析`
   这是 Lance 体系的分析起点，先搞清市场真相、市场分析目的、主观与客观、分析流程。
2. `卷3：交易策略`
   这是最关键的一卷。
   重点先抓：
   `架构概念 -> 架构定义 -> 架构区域 -> 止损 -> 目标 -> 入场 -> 交易管理 -> 交易过程清单`
3. `卷4：你的交易业务`
   重点先抓：
   `资金管理 -> 意外事件管理 -> 目的与目标 -> 交易心理 -> 交易计划 -> 程序手册`

这三卷合起来，才是 Lance 真正对你系统有基础价值的部分。

### 第二批再读

1. `卷1：简介`
2. `卷5：交易者成长`
3. `卷6：总结`

这些适合在你已经进入执行层之后再补。

### 可后读

1. 视频
2. AI 派生的系统设计文档
3. 各种应用版材料

这些都可以后看，主线先以原始六卷文本为准。

## 三书合并后的最优顺序

如果只给你一个总顺序，我建议这样走：

1. `《交易圣经》第1章`
2. `《交易圣经》第2章`
3. `《交易圣经》第7章`
4. `《交易圣经》第8章`
5. `《交易圣经》第9章`
6. `《专业投机原理》第3章`
7. `《专业投机原理》第4章`
8. `《专业投机原理》第5章`
9. `《专业投机原理》第7章`
10. `《专业投机原理》第11章`
11. `《专业投机原理》第13章`
12. `YTC 卷2`
13. `YTC 卷3`
14. `YTC 卷4`
15. 再回头补三书的第二批章节

## 最后裁决

这三本如果要服务“系统基础”，首批只需要抓三件事：

1. `《交易圣经》` 抓系统三柱与流程
2. `《专业投机原理》` 抓趋势、风险、情绪纪律
3. `Lance` 抓架构、入场、止损、管理

一句话收口：

**先用《交易圣经》搭骨架，再用《专业投机原理》压实底层逻辑，最后用 Lance 把执行颗粒度做出来。**
