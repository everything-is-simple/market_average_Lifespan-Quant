# 价格行为四书-首批必读章节顺序

日期：2026-03-29

## 一句话先讲

这四本不要四线并读。

最有效的顺序是：

1. `许佳冲《裸K线交易法》` 先立中文执行直觉
2. `Bob Volman《外汇超短线交易》` 再立结构与突破节奏
3. `Lance / YTC` 把流程、架构、管理收束成系统
4. `Al Brooks` 只做定点攻克，不要全书硬啃

## 第一部：许佳冲《裸K线交易法》先读顺序

### 首批必读

1. `第3章 交易系统详解`
   先把 `标的、仓位、方向、入场点、止损、止盈、对策、后手` 这八件事钉住。
2. `第5章 关键位`
   这是这本书最实用的地基，先立关键位，再谈信号。
3. `第6章 裸K线交易法(Price Action)`
   重点抓：
   `信号研判 -> 盈利空间 -> 入场和止损方法`
4. `第9章 复盘`
   这章很值钱，能把前面的信号和执行变成训练闭环。
5. `第10章 交易心理`
   用来防止把前面的技术又读成“凭感觉交易”。

### 第二批再读

1. `第4章 Pinbar`
2. `第8章 金字塔加仓法`
3. `第7章 实战应用`

这几章不是不重要，而是应该在你已经立住关键位和执行规则后再读。

### 可后读

1. `第1章 二八定律`
2. `第2章 K线`

这些更适合当热身读物，不是你现在最缺的主干。

## 第二部：Bob Volman《外汇超短线交易》先读顺序

### 首批必读

1. `第5章 概率原理`
   不先吃这章，后面的超短结构很容易读成形态崇拜。
2. `第6章 结构`
   这是整本书的门槛章，也是最重要的基础章。
3. `第8章 首次突破`
4. `第9章 二次突破`
5. `第10章 箱体突破`
6. `第11章 区间突破`
7. `第14章 临界点技术`

这几章合起来，基本就是 Bob 的核心语法。

### 第二批再读

1. `第7章 双十字星突破`
2. `第12章 区间内突破`
3. `第13章 高级区间突破`
4. `第15章 不利市场条件`

这些是很好的精进章，但不必最先读。

### 可后读

1. `第1章 交易的货币对`
2. `第2章 交易笔数走势图`
3. `第3章 把超短线交易当做职业`
4. `第4章 目标位、止损位和订单`
5. `第16章 交易量`
6. `第17章 最后的告诫`

这些有用，但更像边界、背景和补充。

## 第三部：Lance / YTC 先读顺序

### 首批必读

1. `卷2：市场和市场分析`
   重点抓：
   `市场的真相 -> 我们如何获利 -> 有效策略和系统 -> 市场分析目的`
2. `卷3：交易策略`
   这是 Lance 的核心。
   重点抓：
   `架构的概念 -> 架构定位规则 -> 架构定义 -> 当价格进入架构区时 -> 止损 -> 目标 -> 入场 -> 交易管理 -> 交易过程清单`
3. `卷4：你的交易业务`
   重点抓：
   `资金管理 -> 意外事件管理 -> 目标 -> 交易心理 -> 交易计划 -> 程序手册`

### 第二批再读

1. `卷1：简介`
2. `卷5：交易者成长`
3. `卷6：总结`

这些适合在主干建立后再补。

### 尤其要回看的章节

1. `卷3 第5章 交易示例`
   这里的 `BPB / PB / BOF / TST` 最适合在你吃完前面的架构和管理后回看。

## 第四部：Al Brooks 先读顺序

### 注意

Al Brooks 不要按“从第一页到最后一页”硬顶。

正确方式是：

`定点攻克`

### 首批必读

1. `趋势篇 第1章 价格行为的频谱`
   先建立“趋势到交易区间是连续频谱”这个总观念。
2. `趋势篇 第3章 突破、交易区间、测试和反转`
   这是他整套语言最关键的总入口之一。
3. `趋势篇 第4章 棒线基础：信号棒、入场棒、架构和K线形态`
4. `趋势篇 第10章 二次入场`
5. `趋势篇 第13章 趋势线`
6. `趋势篇 第17章 水平线：波段点和其他关键价位`
7. `趋势篇 第19章 趋势很强的征兆`
8. `趋势篇 第20章 两条腿形态`

这几章最适合拿来给前三本做“语法校准”。

### 第二批再读

1. `趋势篇 第2章 趋势棒、十字星棒和高潮`
2. `趋势篇 第5章 反转棒`
3. `趋势篇 第8章 棒线收盘价的重要性`
4. `趋势篇 第11章 迟到的和错过的入场`
5. `趋势篇 第12章 形态演变`

这些更适合你在已经有大量盘感后回头细抠。

### 可后读

1. `趋势篇 其余章节`
2. `区间篇、反转篇整本`

不是说不重要，而是现在不该全压过来，不然很容易再次“十年没看懂”。

## 四本合并后的最优顺序

如果只给你一个总顺序，我建议这样走：

1. `许佳冲 第3章`
2. `许佳冲 第5章`
3. `许佳冲 第6章`
4. `Bob 第5章`
5. `Bob 第6章`
6. `Bob 第8/9/10/11章`
7. `Bob 第14章`
8. `Lance 卷2`
9. `Lance 卷3`
10. `Lance 卷4`
11. `Al 趋势篇 第1/3/4/10/13/17/19/20章`
12. 再回头补四本的第二批章节

## 最后裁决

这四本如果要服务你的系统，优先级应该是：

1. `许佳冲` 负责中文落地
2. `Bob` 负责结构和突破节奏
3. `Lance` 负责流程、架构和管理收束
4. `Al Brooks` 负责原典语法校准

一句话收口：

**先用许佳冲和 Bob 把动作练出来，再用 Lance 把动作组织成系统，最后用 Al Brooks 去解释这些动作背后的语言。**
