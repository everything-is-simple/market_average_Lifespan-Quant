# 《1000%男人》首批必读章节与应跳读章节

日期：2026-03-28

## 一句话先讲结论

这本书不要按标题读，也不要从 `5分钟交易` 开始读。

最省时间的读法是：

`先抓 Part1 的日K/30分钟主干 + Part1 的避亏等待 + Part2 的心态/攻破法/资金管理`

再回头看 `5分钟、比赛故事、具体买卖谱`。

## 首批必读

### 1. Part1 前言

先读原因：

- 菲阿里在这里先把自己的交易观讲明白了
- 明确说自己不是靠神秘方法，而是把常见定式加上现实判断来用
- 直接把“半年11倍”从神话拉回到“薄利积累 + 风险处理”
- 明确交代了起始资金、单次风险、隔夜风险边界

这部分是整本书最重要的“使用说明书”。

### 2. Part1 第2章《利用日K线、30分钟K线作转向交易》

这是菲阿里部分真正的主干。

先读原因：

- 它已经不是最噪音的超短，而是开始把 `日线方向 -> 30分钟关键位 -> 开平仓` 串起来
- 能清楚看到他如何判断趋势、如何选汽油/煤油、如何处理无利润隔夜
- 这是最接近“可迁移交易骨架”的部分

如果只读菲阿里一章，优先读这章。

### 3. Part1 第4章《避免亏损、等待时机》

这是最容易被低估、但其实最值钱的一章。

先读原因：

- 它讲的不是如何更猛地赚钱，而是如何在 `9·11` 之后、在箱体环境中、在收益已很高时避免把成果吐回去
- 能看到他如何主动降速、如何以差价/套利为主、如何把“不做”当成正式动作
- 这章比“神奇买点”更接近成熟交易者的核心能力

如果你只想抓“真正的宝贝”，这一章优先级非常高。

### 4. Part2 第1章《在交易市场上赚钱应具备的心态》

这是炭谷部分最好的开场章。

先读原因：

- 它直接把“高手靠预测”这个幻觉打掉
- 反复强调：赚钱不在于猜涨跌，而在于结算、止损、资金管理
- `从欲望到系统` 这一段很值得进卡片系统

这一章不是鸡汤，而是后半本方法论的总开关。

### 5. Part2 第3章《3点开始30分钟石油攻破法》

这是炭谷部分最值得读的实操章节。

先读原因：

- 它是明确的规则化方法，不是泛泛故事
- 有具体的进场、平仓、统计结果
- 能看到“系统不是神奇预测，而是规则 + 执行 + 统计结果”

这章是后半本最像“系统教材”的部分。

### 6. Part2 第5章《资金管理》

这章必须进入首批必读。

先读原因：

- 书里最硬的一句就在这章：`资金管理使市场交易从投机走向实业`
- 它明确说清楚：没有系统，就谈不上资金管理
- 也明确承认：系统会失效，历史统计不是保险单，先考虑不破产

如果你把这章跳过，就等于只读了半本书。

## 第二批再读

### 1. Part1 第1章《利用5分钟K线交易图的转向交易》

这章不是没价值，但不适合最先读。

原因：

- 噪音更大
- 更依赖临场判断
- 容易把注意力带到“超短反应”上，而不是方向、结构、风险

正确读法：

- 在读完 Part1 第2章和第4章之后，再把它当成“早期打法样本”和“错误学样本”来看

### 2. Part1 第3章《不规范的差价·套利》

这章值得读，但不适合最早读。

原因：

- 它建立在你已经理解菲阿里前面几章交易骨架之后
- 如果一开始就读，容易只看见术而看不清基底

正确读法：

- 先立住单向交易、持仓、隔夜、等待时机
- 再回来理解他为什么会转向差价和套利

### 3. Part2 第2章《玉米当日交易大作战》

这章可以读，但优先级低于石油 30 分钟攻破法。

原因：

- 它更像规则化日内交易样本
- 价值在于“逆出价”“规则化平仓”“统计检验”
- 但和你当前阅读主线的贴合度，不如 Part2 第3章和第5章

正确读法：

- 放在心态、30分钟攻破法、资金管理之后再读

### 4. Part2 第4章《在ROBBINS-TAICOM公司举行的“期货冠军”大赛中使用的手法》

这章很有趣，但更多是“比赛特定环境 + 当时市场漏洞 + 临场经验”。

原因：

- 作者自己都明确说这些办法后来很多已经作废
- 更像研究材料，而不是今天可直接照搬的主干

正确读法：

- 当作“市场结构变化会淘汰旧优势”的案例来读

## 可以直接跳读或只抽读的

### 1. Part3《两位冠军的买卖谱》

这部分非常珍贵，但不适合开卷就啃。

原因：

- 信息量大
- 结构更像原始流水，而不是解释性文字
- 没有前面的方法框架，读起来很容易只剩“好多单子”

正确读法：

- 前面章节读完后，再把它当验证材料和原始证据来抽读

### 2. 前面的“冠军形成”“比赛背景”“序言”类内容

这些可以快速浏览，不必深陷。

原因：

- 它们负责建立背景和氛围
- 但对你真正有迁移价值的，不是“冠军传说”，而是交易骨架和资金管理

## 最省时间的阅读顺序

如果你只想用最少时间抓住这本书的精华，建议按这个顺序走：

1. `Part1 前言`
2. `Part1 第2章`
3. `Part1 第4章`
4. `Part2 第1章`
5. `Part2 第3章`
6. `Part2 第5章`
7. `Part1 第3章`
8. `Part1 第1章`
9. `Part2 第2章`
10. `Part2 第4章`
11. `Part3 买卖谱抽读`

## 最后裁决

这本书最正确的打开方式，不是：

- 先看 5 分钟
- 先看 11 倍神话
- 先看比赛传奇

而是：

- 先看方向和结构
- 再看避亏与等待
- 再看系统化方法
- 最后回头看超短、比赛细节和流水

一句话收口：

**先读 `Part1 第2章 + Part1 第4章 + Part2 第1/3/5章`，这才是《1000%男人》真正该先拿走的骨头。**
