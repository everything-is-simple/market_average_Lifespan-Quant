  `林辉太郎这三本，先读哪本，哪本放后面，每本又该先抓哪些章节。`

---

## 1. 先给总顺序

书目顺序建议：

1. `《期货市场的技术》`
2. `《波段交易法》`
3. `《价差交易入门》`

一句话解释：

`先读主干体系，再读股票版训练书，最后读价差专项教材。`

为什么不是先读《价差交易入门》：

1. 它更像林系实践教材
2. 但没有《期货市场的技术》那么接近总骨架
3. 先读它容易把林系误看成“只是价差交易”

---

## 2. 第一本先读：《期货市场的技术》

### 首批必读顺序

1. `第3章 建仓的技术`
2. `第6章 交易基础练习法`
3. `第4章 仓位的转换和头寸的对冲`
4. `第5章 走出期货外行的阶段`
5. `第7章 头寸操作与平衡`

### 为什么这样排

1. `第3章` 先把建仓法则、顺逆势、资金和计划性立起来
2. `第6章` 再把试验头寸、主力头寸、分批投入、失败应对吃下来
3. `第4章` 再进入价差、对冲、换月这些林系核心
4. `第5章` 用来理解林系的技术观和成长路径
5. `第7章` 最后看头寸平衡和成功/失败整理

### 这些章重点抓什么

1. `第3章`：
   `建仓法则、顺势/逆势、计划性、节奏`
2. `第6章`：
   `试验头寸、主力头寸、分批建仓、失败应对、对冲与平仓`
3. `第4章`：
   `价差运动、对冲头寸、仓位转换`
4. `第5章`：
   `什么是期货技术、资金和时间、概率、提高的阶段`
5. `第7章`：
   `头寸整理、试验头寸重复建仓、过早平仓、失败原因`

### 第二梯队章节

1. `第1章 所谓的价格波动`
2. `第2章 月份的选择`
3. `第8章 职业炒手们的交易谱`

说明：

1. `第1章` 和 `第2章` 是理论背景，值得读，但不如第3、4、6章直接
2. `第8章` 很精彩，但更适合在你已懂规则后再看

### 可以后读或跳读的部分

1. 大量交易谱细节
2. 具体品种的历史背景
3. 附录和图表目录

原则：

- `先抓规则，不逐条啃案例数字`

---

## 3. 第二本再读：《波段交易法》

### 首批必读顺序

1. `第二部：现段交易法`
2. `第四部：几位打算炒股为生伙伴们的聚会`
3. `第三部：统计与现实`
4. `第一部：让我们看看成功的交易者`

如果按更细的章节入口抓，建议优先读：

1. `二、关于炒股的工具`
2. `八、选股就从自己熟悉的股票当中选`
3. `二十、进入未曾体验领域的恐惧感`
4. `五、资金账户中要留出宽裕资金`
5. `八、不要一年到头炒股票`
6. `十、该放弃的时候必须懂得放弃`
7. `十一、平仓——退出游戏`
8. `十二、几个炒股成功人士的实例`

### 为什么这样排

1. `第二部` 最接近操作法主干
2. `第四部` 最接近“如何做交易者”这一层
3. `第三部` 是补充性的统计和现实反思
4. `第一部` 更像故事入口，可后读

### 读的时候重点抓什么

1. 周期、工具、熟悉股票池
2. 不要过度迷信图形
3. 给资金和情绪留空间
4. 空仓休息和平仓退出
5. 独立思考，不追两个兔子

### 可以后读或跳读的部分

1. 成功者故事的长篇细节
2. 某些风格鲜明的人物段子
3. 过于散文化的劝诫段落

原则：

- `把它当训练营，不当故事书`

---

## 4. 第三本最后读：《价差交易入门》

### 首批必读顺序

1. `第4章 风险管理和价差交易的手法`
2. `第2章 备齐价差交易的工具`
3. `第1章 价差交易与期货市场`
4. `第9章 季节性的价差`
5. `第5章到第8章（四种形态）`

### 为什么这样排

1. `第4章` 是这本书最值钱的硬骨架
2. `第2章` 让你先明白工具为什么重要
3. `第1章` 再补足价差交易的基本概念
4. `第9章` 是很有价值的专项补充
5. `第5-8章` 的四种形态更像实践样本，应放在后面

### 读的时候重点抓什么

1. 赢大亏小和最后底线
2. 工具：砖形格、头寸记录本、头寸卡、学习笔记
3. 价差变化本身才是判断材料
4. 顺势和逆势要按性格与方法选择
5. 季节性价差不能脱离资金能力

### 可以后读或跳读的部分

1. 具体实例的每个盈亏数字
2. 附录术语表的大部分条目
3. 讲座式附录中的鼓动性文字

原则：

- `把它当价差操作说明书，不当发财圣经`

---

## 5. 最省时间的总读法

如果你只想最快把林系读对，建议：

1. 先读《期货市场的技术》的 `第3、6、4、5、7章`
2. 再读《波段交易法》的 `第二部 + 第四部`
3. 最后读《价差交易入门》的 `第4、2、1、9章`

这样读完后，你基本就不会再把林辉太郎误读成：

- 一个图形派大师
- 一个只讲价差的专门派
- 一个只会讲心法的老派作家

---

## 6. 最终收口

林系真正该先读的，不是某个传奇案例，而是它如何把：

`建仓 -> 试验头寸 -> 主力头寸 -> 对冲 -> 平仓 -> 休息`

串成一套交易技术。

一句话压缩：

`先读《期货市场的技术》学骨架，再读《波段交易法》学训练，再读《价差交易入门》学专项。`
