  1. 先系统层

## 1. 先读规则

这 20 篇不是“最强策略 Top20”，而是“最能帮你建立正确阅读骨架的 Top20”。

阅读时统一带着四个问题：

1. 它先判环境，还是先找机会
2. 它有没有把选股、择时、仓位、退出串成闭环
3. 它是系统表达，还是平台脚本堆叠
4. 它能补什么直觉，不能定义什么概念

一句话：

`先学它们怎么组织问题，不先学它们怎么吹收益。`
