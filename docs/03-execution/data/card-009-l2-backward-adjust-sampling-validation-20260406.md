# CARD-009 L2 后复权抽样验证（待 L1 灌入后）/ 2026-04-06

## 目标

验证 `market_base`（L2）的后复权价格计算正确性：
1. 抽样验证后复权价与通达信软件显示一致
2. 确认 gbbq 除权除息因子链的完整性和正确性
3. 特别关注分红、送股、配股三类权益事件

## 前置条件

- [ ] card-001 完成：`raw_market` + `market_base` 已填充
- [ ] 通达信软件可用（`H:\new_tdx64`），用于人工对照

## 执行步骤

1. 随机抽取 20 只股票（覆盖沪市/深市/北交所/ST/退市）
2. 对每只股票：
   - 导出 `stock_daily_adjusted`（backward）最近 5 年的 OHLCV
   - 在通达信软件中查看同标的同时段后复权价
   - 逐日对比：收盘价差异 < 0.01 元
3. 特别验证权益事件日：
   - 分红除权日前后价格连续性
   - 送股/转增日前后股数变化反映在价格上
   - 配股日的处理
4. 验证 gbbq 因子链：
   - `adjust_factor` 表的因子值在权益事件日有跳变
   - 因子值的累乘路径与通达信一致

## 验收标准

1. 20 只股票 × 5 年日线，后复权价与通达信差异 < 0.01 元（允许四舍五入差异）
2. 权益事件日前后价格连续性通过（无断崖跳变）
3. 北交所股票和 ST 股票均通过
4. 无 NULL 值出现在 OHLCV 核心字段

## 产出物

- evidence（抽样对比表 + 权益事件日截图）
- record + conclusion
